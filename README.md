# iPhone voice input + browser translate

https://tomosud.github.io/iphone_trans/

## 使い方A　MEMO

webブラウザブラウザだけでMEMOを作成するToolです。
MEMOはローカルに一時保存（indexdDB）されます。

## 使い方B　リアルタイム翻訳

iphoneの音声入力で英語の音声を入力し、webブラウザのgoogle翻訳の機能を使うことでリアルタイム翻訳のように使うことができます。

以下のように、google翻訳と音声入力（EN）を有効にしてください。

翻訳を有効。テキストフィールドを選んで、マイク入力（EN）を選択。
